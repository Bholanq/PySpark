# Workspace
A **Databricks Workspace** is the **collaborative environment where users interact with Databricks**
Don't cost money

- UI where users interact with the Databricks Web Application
- Contains:
	- Cluster
	- Notebooks
	- Jobs
	- Repos

```
Company Databricks Account
│
├── Dev Workspace
├── QA Workspace
├── UAT Workspace
└── Production Workspace
```
